Proyecto AnimePe

En este proyecto se desarrolló una aplicación web de anime, donde los usuarios pueden ver información sobre diferentes animes, incluyendo sinopsis, géneros y episodios. La aplicación utiliza la API de Jikan para obtener datos de MyAnimeList y está construida con Angular y Bootstrap para un diseño responsivo y atractivo.
