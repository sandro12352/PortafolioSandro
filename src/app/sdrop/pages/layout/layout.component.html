Proyecto AnimePe

En este proyecto se desarrolló una aplicación web de anime, donde los usuarios pueden ver información sobre diferentes animes, incluyendo sinopsis, géneros y episodios. Proyecto Osana

En este proyecto se desarrolló una aplicación web para una despacho de arquitectos, donde los usuarios pueden ver información sobre diferentes proyectos arquitectónicos, incluyendo imágenes, descripciones y detalles técnicos. La aplicación utiliza una base de datos MySQL para almacenar la información y está construida con Angular y Bootstrap para un diseño responsivo y atractivo.

También se uso un formulario con la API de Meta donde al llenar el formulario y dar en Enviar automaitacamente los datos escritos por el usuario llegan al whatsapp de la empresa.

Proyecto Colegio

Aplicación desarrollada en Python (PyQt5) con base de datos en MySQL, pensada para colegios. Permite gestionar matrículas, pagos y datos de alumnos desde una interfaz intuitiva. Automatiza tareas administrativas, mejora el control de la información y optimiza los procesos escolares.

La aplicación incluye funcionalidades como búsquedas filtradas, validación de datos, generación de reportes y una interfaz amigable para facilitar su uso al personal administrativo. Gracias a su estructura modular y conexión a base de datos, se asegura la integridad y disponibilidad de la información.
